# pay_password_edittext
- 高仿支付宝/微信付款页面
- 效果图

- 主要学习和记录自定义 View 的 onDraw() 中， canvas 和 paint 的使用。
## 涉及到知识点
- canvas:
    - 绘制矩形
    - 绘制圆角矩形
    - 绘制圆
    - 绘制线
- paint:
    - 设置颜色
    - 设置 stroke
    - 设置 strokewidth
    - 设置 fill
